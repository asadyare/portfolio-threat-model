# Risk Register

Security Detection and Reporting

Each identified risk is mapped to one or more detection or reporting mechanisms.

## Detection Sources

CI workflows in portfolio-ci-cd-security
Daily security scans in portfolio-daily-security

## Alerting

Critical risks generate automatic GitHub Issues
Alerts link to workflow evidence and SARIF results

## Reporting

Weekly security reports summarise scan execution and findings
Reports provide governance level visibility without manual effort

## Validation

GitHub Actions logs
GitHub Security tab
Security alert and report issues

See risk-register.md for full table.
